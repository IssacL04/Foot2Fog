# Foot2Fog
A simple python script to convert multiple CSVs to FoW compatible GPX with continuous points.
The script is altered based on:https://github.com/XFang95/CSVtoGPX , thanks to the original author!
Usage: put the CSV files to be converted in the ../input folder and run python csvtogpx.py, the output results will be listed in the ../output folder, which can be directly imported in the Fog of World app.
Currently support: Flight CSV on Variflight platform & Output backup data CSV from Footprint app.
